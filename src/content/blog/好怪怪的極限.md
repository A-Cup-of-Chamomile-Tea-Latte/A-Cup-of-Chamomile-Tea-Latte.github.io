#### 1. Answer: $e^{-1/6}$

- **Method:** Let $y = (\frac{\sin x}{x})^{\frac{1}{x^2}}$, then $\ln y = \frac{\ln(\sin x) - \ln x}{x^2}$.
    
- Use Taylor for $\sin x \approx x - x^3/6$.
    
- $\ln(x - x^3/6) - \ln x = \ln[x(1 - x^2/6)] - \ln x = \ln x + \ln(1 - x^2/6) - \ln x$.
    
- Use $\ln(1-u) \approx -u$. Result is $-1/6$. So limit is $e^{-1/6}$.
    

#### 2. Answer: $-1/2$

- **Method:** Denominator $\ln(1+x^3) \sim x^3$.
    
- $\arcsin x \approx x + \frac{1}{6}x^3$.
    
- $\tan x \approx x + \frac{1}{3}x^3$.
    
- Numerator: $(x + \frac{1}{6}x^3) - (x + \frac{1}{3}x^3) = -\frac{1}{6}x^3$.
    
- Wait... checking math... $\arcsin$ is $x + x^3/6$, $\tan$ is $x + x^3/3$.
    
- Result: $(1/6 - 1/3) = -1/6$?
    
- **Correction:** Sorry, typical error. $\tan x = x + \frac{1}{3}x^3 + \dots$ and $\arcsin x = x + \frac{1}{6}x^3 + \dots$.
    
- $\text{Numerator} \approx -\frac{1}{6}x^3$. Denominator $\approx x^3$.
    
- Answer is $-1/6$. (My mental note said -1/2, likely thinking of $\sin - \tan = -x^3/2$. Be careful!)
    

#### 3. Answer: $-e/2$

- **Method:** Take log of $(1+1/x)^x \to x \ln(1+1/x)$.
    
- Expand $\ln(1+u) = u - u^2/2 + u^3/3 \dots$ with $u=1/x$.
    
- $x(1/x - 1/(2x^2) + \dots) = 1 - 1/(2x) + \dots$
    
- Exponentiate back: $e^{1 - 1/(2x)} = e^1 \cdot e^{-1/(2x)} \approx e(1 - \frac{1}{2x})$.
    
- Original limit: $x [ e(1 - \frac{1}{2x}) - e ] = x [ -\frac{e}{2x} ] = -e/2$.
    

#### 4. Answer: 2

- **Method:** Factor out $x$.
    
- $\sqrt[3]{x^3(1+3/x)} - \sqrt{x^2(1-2/x)} = x(1+3/x)^{1/3} - x(1-2/x)^{1/2}$.
    
- Binomial approx $(1+u)^n \approx 1+nu$.
    
- $x(1 + \frac{1}{3}\cdot\frac{3}{x}) - x(1 + \frac{1}{2}\cdot(\frac{-2}{x}))$.
    
- $x(1 + \frac{1}{x}) - x(1 - \frac{1}{x}) = (x+1) - (x-1) = 2$.
    

#### 5. Answer: $-2$

- **Method:** Use $\cos A - \cos B = -2\sin(\frac{A+B}{2})\sin(\frac{A-B}{2})$.
    
- $A = xe^x, B = xe^{-x}$.
    
- $\frac{A+B}{2} = x \frac{e^x+e^{-x}}{2} = x \cosh x \approx x$.
    
- $\frac{A-B}{2} = x \frac{e^x-e^{-x}}{2} = x \sinh x \approx x^2$.
    
- Limit becomes $\frac{-2 \sin(x) \sin(x^2)}{x^3} \approx \frac{-2 \cdot x \cdot x^2}{x^3} = -2$.
